Порядок установки
1. Установить MicrosoftSpeechPlatformSDK
2. Установить SpeechPlatformRuntime
3. Установить MSSpeech_TTS_ru-RU_Elena (для того, чтобы программа "говорила" русским)
4. Установить MSSpeech_SR_ru-RU_TELE (для того, чтобы программа распознавала русский)
5. После этого можно запускать программу и начинать использование

Если требуются другие языки для произношения программой и распознавания:
https://www.microsoft.com/en-us/download/details.aspx?id=27224 здесь после нажатия на Download открывается список всех возможных языков.
Те, что начинаются на "MSSpeech_TTS" (text to speech) - для синтеза речи.
Те, что начинаются на "MMSSpeech_SR" (speech recognition) - для распознавания речи.


Для конфигурации под себя ознакомьтесь с файлом config_ru.txt.
